# PythonClientServer2020
Simple client-server app

Приложение должно получать заказ от покупателя -- перечень товаров и их количество, так же создавать заказ с id и состоянием которое отражает информацию о том, где сейчас заказ. Ну и имеет функцию печатать что есть на складе сейчас.
Второе приложение -- дает возможность обновлять состояние заказов и склада, оставлять письма заказчикам.
Ну и сам сервер, который это все обрабатывает.
Стоит заметить, что в коде много привязок к конкретным файлам json. Это связано с тем, что работать с бд мы не умеем, а хранить информацию хочется.
